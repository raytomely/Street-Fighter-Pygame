SF Converter is a tool to convert characters from various 
fighting game engines to the SIBM format and edit them.
in the characters folder there is characters that I have converted 
from various fighting game engines.
it include:
- Charlce and Stickw from KOF91 fighing engine.
- Spiderman and Hayato from UFGE (Universal Fighting Game Engine).
- and from MUGEN we have: Goku, Vegeta,Gohan, Toguro and Hiei.
